# ♟️ Pro Chess Game

A modern and interactive Chess Game built using HTML, CSS, and JavaScript. This project provides a fully functional chessboard with drag-and-drop gameplay, legal move validation, move history tracking, and game status updates.

## 🚀 Features

* Drag-and-drop chess pieces
* Legal move validation
* Check detection
* Checkmate detection
* Draw detection
* Move history tracking
* Responsive design
* Modern user interface
* Powered by Chess.js and Chessboard.js

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chess.js
* Chessboard.js
* jQuery

## 📂 Project Structure

ChessGame/

├── index.html

├── style.css

└── script.js

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
6. Start playing chess in your browser.

## 🎮 Gameplay

* Drag and drop pieces to make moves.
* Only legal moves are allowed.
* The game automatically detects:

  * Check
  * Checkmate
  * Draw situations
* Move history is displayed below the board.
